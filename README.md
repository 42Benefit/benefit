# 프로젝트 소개

benefit는 42서울 카뎃들을 위한 혜택들을 모아놓은 사이트입니다.

제휴기업들이 `은하수를 여향하고 있는 히치하치하이커`들에게 제공하는 혜택과 이용밥법을 참고해보세요! 

[사이트주소](https://benefit.42seoul.link/)

## 기능
- 다양한 혜택 정보를 한눈에 확인할 수 있습니다.
- 혜택 정보를 검색을 통해 확인할 수 있습니다.
